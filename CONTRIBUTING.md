# Contributing to EduPro Academy

Thank you for your interest in contributing! This project aims to create a professional education platform.

## How to Contribute

### 1. Fork the Repository

Click the "Fork" button at the top right of this page.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/Academy-Website.git
cd Academy-Website
```

### 3. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Changes

- **Fix bugs** in existing files
- **Add new pages** following the existing design pattern
- **Improve styling** in `css/style.css`
- **Add new courses** in `js/main.js`
- **Fix typos** or improve documentation

### 5. Test Your Changes

```bash
# Open index.html in your browser
start index.html
# or
xdg-open index.html
```

### 6. Commit and Push

```bash
git add .
git commit -m "Add your descriptive commit message"
git push origin feature/your-feature-name
```

### 7. Create a Pull Request

Go to your fork and click "New Pull Request".

## Contribution Ideas

- [ ] Add more course data to `js/main.js`
- [ ] Create new pages (blog, pricing, FAQ)
- [ ] Improve accessibility (ARIA labels, keyboard navigation)
- [ ] Add new language support
- [ ] Create additional UI components
- [ ] Improve mobile responsiveness
- [ ] Add loading skeletons
- [ ] Implement dark/light mode toggle
- [ ] Add animations
- [ ] Write unit tests

## Code Style

- Use semantic HTML5
- Follow Bootstrap 5 conventions
- Use CSS variables from `css/style.css`
- Add AOS animations for scroll effects
- Write clean JavaScript with comments

## Questions?

Open an issue if you have questions!
